一. 支持Vim  
-----
1 . 安装 keyboard-localization
> apm install keyboard-localization

2 .
>安装 vim-mode， 和 ex-mode

3 . 配置键盘
> Edit => preferences => settings => Packages -->
> keyboard-localization => settings => en_GB
